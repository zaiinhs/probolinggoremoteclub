# Website Copy MVP Probolinggo Remote Club

## Tujuan Dokumen

Dokumen ini berisi draft copywriting untuk halaman-halaman utama website Probolinggo Remote Club. Copy disusun agar bisa langsung dipakai sebagai dasar desain, landing page, dan implementasi awal website.

Gaya bahasa yang digunakan:

1. Hangat dan terbuka.
2. Jelas dan mudah dipahami.
3. Tidak terlalu formal, tetapi tetap rapi dan kredibel.
4. Menekankan bahwa komunitas ini gratis, non-profit, dan inklusif.

## Brand Voice Ringkas

Karakter komunikasi Probolinggo Remote Club:

1. Lokal tapi modern.
2. Ramah dan tidak eksklusif.
3. Relevan untuk pekerja remote lintas profesi.
4. Mengundang, bukan mengintimidasi.
